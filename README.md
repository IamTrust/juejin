# 仿掘金官网

## 后端接口部分

后端接口`API`：

`http://123.60.213.2:9088/juejinserver/index_data/find_article/${current}/${limit}`

参数：`current`为当前页，`limit`为每页记录数。

## 前端部分

### 启动

#### 进入`juejin-app`目录

```
cd juejin-app
```

#### 安装依赖

```
npm install
```

#### 启动服务器

```
npm run serve
```
